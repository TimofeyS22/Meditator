# Политика конфиденциальности приложения Meditator

Дата вступления в силу: 29 марта 2026 г.

## 1. Общие положения

Настоящая Политика конфиденциальности (далее — Политика) определяет порядок обработки и защиты персональных данных пользователей мобильного приложения Meditator и сайта mymeditator.ru (далее — Сервис).

Оператор персональных данных: ИП Семикин Т.А. (далее — Оператор).

Email: support@mymeditator.ru

Используя Сервис, вы соглашаетесь с условиями настоящей Политики. Если вы не согласны с условиями, пожалуйста, не используйте Сервис.

## 2. Какие данные мы собираем

### 2.1 Данные, предоставляемые пользователем

- **Данные аккаунта**: адрес электронной почты, имя (по желанию), хэшированный пароль
- **Данные профиля**: выбранные цели медитации, уровень стресса, предпочтения по длительности и времени практик
- **Данные об использовании**: записи настроения, история медитаций, статистика практик, растения в саду покоя
- **Сообщения чату с AI-компаньоном**: текст сообщений

### 2.2 Данные, собираемые автоматически

- **Технические данные**: тип устройства, версия ОС, IP-адрес (для обеспечения безопасности)
- **Данные сессий**: длительность медитации, время начала/окончания

### 2.3 Данные, которые мы НЕ собираем

- Геолокация
- Контакты телефонной книги
- Данные банковских карт (обработка платежей осуществляется через ЮKassa)

## 3. Цели обработки персональных данных

Мы обрабатываем данные для:

- Предоставления и персонализации Сервиса, включая AI-рекомендации
- Обработки платежей и управления подписками
- Улучшения качества Сервиса и разработки новых функций
- Обеспечения безопасности и предотвращения злоупотреблений
- Отправки сервисных уведомлений (с вашего согласия)

## 4. Правовые основания обработки

Обработка персональных данных осуществляется на основании:

- Согласия субъекта персональных данных (ст. 6 152-ФЗ)
- Исполнения договора (Пользовательское соглашение)
- Законных интересов Оператора (безопасность, улучшение сервиса)

## 5. AI и обработка данных

Для генерации персонализированных медитаций мы передаём контекст вашего настроения и предпочтений сторонним API (OpenAI). Передаваемые данные не содержат вашего имени, email или иных прямых идентификаторов. Мы не передаём данные третьим лицам для маркетинговых целей.

## 6. Хранение и защита данных

- Данные хранятся на защищённых серверах на территории Российской Федерации
- Шифрование соединения (TLS/SSL)
- Хэширование паролей (bcrypt)
- JWT-аутентификация с ротацией токенов
- Rate limiting для защиты от атак
- Регулярное обновление программного обеспечения

## 7. Передача данных третьим лицам

Мы передаём данные только:

- **ЮKassa** — для обработки платежей
- **OpenAI** — для генерации AI-контента (обезличенные данные)
- **Sentry** — для мониторинга ошибок (технические данные)
- По запросу уполномоченных государственных органов в случаях, предусмотренных законодательством РФ

## 8. Права пользователя

В соответствии с 152-ФЗ вы имеете право:

- Получить информацию об обработке ваших персональных данных
- Потребовать исправления неточных данных
- Потребовать удаления ваших данных (удаление аккаунта)
- Отозвать согласие на обработку персональных данных
- Обжаловать действия Оператора в Роскомнадзор

Для реализации прав обратитесь по адресу: support@mymeditator.ru

## 9. Файлы cookie и аналитика

Сайт mymeditator.ru использует localStorage для хранения токенов авторизации. Мы не используем сторонние трекинговые cookie.

## 10. Срок хранения данных

Персональные данные хранятся в течение всего срока действия аккаунта. При удалении аккаунта данные удаляются в течение 30 дней, за исключением данных, которые мы обязаны хранить в соответствии с законодательством РФ.

## 11. Дети

Приложение не предназначено для лиц младше 12 лет. Мы не собираем сознательно данные детей.

## 12. Изменения Политики

Мы можем обновлять настоящую Политику. О существенных изменениях мы уведомим через Приложение. Продолжение использования Сервиса после изменений означает согласие с обновлённой Политикой.
